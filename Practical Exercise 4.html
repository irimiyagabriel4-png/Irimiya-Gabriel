Responsive Mockups Assignment

1. What screen did you design, and which two versions (mobile and desktop) did you create?
I designed a homepage screen for a simple service/product website.

The homepage includes:
  • Header (logo + navigation)
  • Hero section (headline + call-to-action button)
  • Content section (feature cards)
  • Footer
I created two versions:
  • Mobile version (~375px width)
  • Desktop version (~1440px width)
2. How do elements rearrange or resize between the mobile and desktop versions?
Header:
Desktop: Logo left, navigation links on the right.
Mobile: Navigation hidden in a hamburger menu.

Hero Section:
Desktop: Text and image side-by-side.
Mobile: Content stacked vertically.

Content Section:
Desktop: 3-column grid.
Mobile: Single-column stacked layout.

Footer:
Desktop: Columns.
Mobile: Stacked and center-aligned.

Other Adjustments:
  • Text resized for readability
  • Increased spacing for touch interaction
3. What button and header component did you create in Figma, and how did you demonstrate consistency?
Button Component:
  • Blue background
  • White text ("Get Started")
  • Rounded corners
  • Consistent padding
Header Component:
  • Logo
  • Navigation links
  • Hamburger menu (mobile)
Consistency Demonstration:
  • Used components across multiple screens
  • Editing the component updated all instances automatically
