The home screen displays a list of tasks with checkboxes and an add task button. The task details screen shows more information about a selected task and allows the user to mark it as complete, edit, or delete it.

3. User Flow

The user starts on the Home Screen and views a list of tasks. When the user clicks on a task (e.g., "Buy Food"), they are taken to the Task Details Screen. On the details screen, the user can perform actions or go back to the home screen.
