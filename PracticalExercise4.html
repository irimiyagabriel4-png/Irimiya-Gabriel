Assignment Answers

1. Which app screen did you design?
I designed a simple login screen for a mobile application.

2. How did you apply visual hierarchy?
I made the title "Login" the largest text to draw attention first. The labels such as "Email" and "Password" are medium-sized, while helper text is smaller and lighter in color to avoid distraction.

3. What primary color did you choose and where did you apply it?
I used blue (#3B82F6) as the primary color and applied it to the "Login" button to make it stand out as the main action.

4. Website Evaluation
I evaluated the YouTube website.

Visual Hierarchy: The YouTube logo is large and placed at the top left, and the search bar is centered, making it easy to find and use.

Consistency: The same style of buttons and icons (like, share, subscribe) is used throughout the website, which makes the interface easy to understand.
